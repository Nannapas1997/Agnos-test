# Adnos Project
> วิธีการติดตั้ง
1. git clone https://github.com/Nannapas1997/Agnos-test/tree/master
2. เปิด terminal แล้วพิมพ์ npm install
3. จากนั้น ให้พิมพ์ npm run dev
4. เปิด http://localhost:5173/
5. สำเร็จ
> รายละเอียดการออกแบบ
> 1. ใช้สีส้มเป็นโทนสีหลักเพราะเป็นเกี่ยวกับการกิน สิ่งที่เป็นสิ่งเร้าคือสีส้ม แดง
> 2. hero section เป็นสีส้มเข้ม ประกอบไปด้วย quote เกี่ยวกับการกิน และรูปที่สื่อถึงการกิน
> 3. สามารถค้้นหา filter sorting ได้หมด
> 4. สินค้าต่างๆ มีรายละเอียด รีวิวคะแนน wishlist
> 5. หน้านึงมี 8 รายการ มี pagination
> 6. มี dark mode และ light mode
